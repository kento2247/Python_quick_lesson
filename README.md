# Pythonクイックレッスン

---
## イントロ
世の中のシステム基盤となるLinuxを使いこなし，AIやプログラムと協働することは，知的労働をする上で重要なスキルです．
Pythonはデータ分析，自動化，AI開発など幅広い用途で使われており，これらを扱えるようになることで仕事や研究の幅が大きく広がります．
また，GitやVSCodeなどのツールも併せて使いこなすことで，チーム開発やプロジェクト管理がスムーズになり，生産性や再現性の高い環境を構築できます．

このレッスンでは，Linux環境のセットアップからPythonのインストール，Git/GitHubの基本操作，VSCodeの設定までをカバーし，AIツールを活用するための基盤を整えることを目指します．
最終的には，Pythonを使って機械学習コードを実行できるようになることを目標としています．
